# Portfolio

A simple portfolio website built with React and TypeScript.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
